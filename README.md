# nuxt_chatgpt
该nuxt项目实现了包括了可以添加自己服务器的各种chatgpt，修改自己的后端服务器地址就可以本地部署专属自己的gpt，包括了登录，轮播图，切换gpt等所有功能，还有支付

1.使用npm install 下载

2.npm run dev运行因为我本地服务器过期了所以没有数据你们换成自己的服务器提供资源就好了
